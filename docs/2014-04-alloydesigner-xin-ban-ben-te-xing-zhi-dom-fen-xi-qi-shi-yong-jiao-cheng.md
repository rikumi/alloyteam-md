DOM 分析器是 AlloyDesigner 在 V1.1.0 版本中引入的新特性  
DOM 分析器有以下几个特点

-   显示文档中的元素与属性
-   属性使用 JSON 数据直观表示
-   显示元素在文档中的位置
-   独有的元素预览功能
